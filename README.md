# 🚗 AvtoElonUz — Car Listing Platform

**AvtoElonUz** is an online platform for buying and selling cars.
Users can post, browse, edit, and manage car listings.
Built with **Spring MVC** and a plain HTML/CSS/JavaScript frontend.

---

## 🎯 Features

- Post, edit, and delete car listings
- Browse and filter available cars
- View sales and transaction history
- User profile management
- Admin panel for full platform control

---

## 👥 Roles

### 👤 User
- Create, edit, and delete own car listings
- Post cars for sale
- View personal listings and transaction history

### 🛠️ Admin
- View all sales and transaction history
- Manage users
- Monitor and moderate all listings

---

## 🏗️ Architecture — Spring MVC

| Layer | Description |
|---|---|
| Model | Entities and DTOs |
| View | HTML pages (server-side rendered) |
| Controller | Handles HTTP requests and business logic |

Frontend and backend are fully integrated — no separate frontend framework used.
